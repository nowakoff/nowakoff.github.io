Ruby Enumerable Cheat Sheet

Basic Methods
count count(value) count { |x| … } Returns the total count of elements, or the count of elements equal to value, or the number of elements where the block returns “true”.
to_a Returns array of all elements
Iteration
each { |x| … } each_entry { |x| … } Yields all elements.
each_with_index(*args) { |x, i| … } Yields all elements with their indices. args are passed through to #each.
each_cons(n) { |x| … } Yields each possible array of n consecutive elements.
each_slice(n) { |x| … } Yields disjoint slices of n consecutive elements.
cycle(times=nil) { |x| … } Yields all elements repeatedly forever or for specified number of times.
reverse_each { |x| … } Yields all elements in reverse order
Sorting
sort [{ |a, b| … }] Returns array sorted by <=> operator or by block. Default block { |a, b| a <=> b }.
sort_by { |x| … } Returns array sorted by the return value of the block.
each_cons(n) { |x| … } Yields each possible array of n consecutive elements.
max [{ |a, b| … }] Returns maximum element. Default block { |a, b| a <=> b }.
max_by { |x| … } Returns element with maximum block return value.
minmax [{ |a, b| … }] Returns [min, max] using block return value.
